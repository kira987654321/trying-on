# Trying On 0.1.0

这是 Trying On｜试装（正在进行时）的 Windows x64 桌面版发布包。

## 下载和运行

下载本页面 Assets 里的：

```text
trying-on-0.1.0-win-x64.zip
```

然后：

1. 解压 zip
2. 进入 `时装ing-win-x64`
3. 双击 `时装ing.exe`

这个 Release 包可以直接运行，不需要 `restore-exe.bat`。

## 说明

仓库首页绿色 `Code -> Download ZIP` 下载的是 GitHub 源码压缩包，不是正式软件发布包。由于 GitHub 普通仓库对单个大文件有限制，源码压缩包里的 exe 使用了分包恢复方式。

普通用户建议下载本 Release 页面里的 zip，这样最简单。
