# Trying On 0.1.0｜试装 Windows 桌面版

这是 Trying On｜试装（正在进行时）的 Windows x64 桌面版发布包。

## 下载和运行

在本页面 Assets 里下载：

```text
trying-on-0.1.0-win-x64.zip
```

然后：

1. 解压 `trying-on-0.1.0-win-x64.zip`
2. 进入 `时装ing-win-x64`
3. 双击 `时装ing.exe`
4. 打开右上角 `设置`
5. 填写 `Base URL`、`API Key` 和模型名称
6. 保存配置后上传商品图并生成

这个 Release 包可以直接运行，不需要 `restore-exe.bat`。

## 这个版本包含什么

- Windows 桌面端服装电商出图工具
- 商品图上传和素材复用
- 多平台模板选择
- 男装、女装模板体系
- 批量生成队列
- 本地结果历史
- 单张下载和批量导出

## API 配置说明

软件支持 OpenAI-compatible image generation API。请在软件设置里填写自己的 `Base URL`、`API Key` 和模型名称。

本项目不会在 GitHub 上固定写死某个服务商 URL，也不会公开你的 API 密钥。

## 下载建议

普通用户建议下载本 Release 页面里的 zip。仓库首页绿色 `Code -> Download ZIP` 下载的是源码压缩包，不是正式软件发布包。
