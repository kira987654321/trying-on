# Trying On

试装（正在进行时）Windows 桌面版。

## 直接下载

点击页面右上方绿色 `Code` 按钮，然后点击 `Download ZIP`，就会下载本仓库里的完整桌面版文件夹。

下载后解压，进入：

```text
时装ing-win-x64
```

先双击：

```text
restore-exe.bat
```

它会把程序启动文件恢复成：

```text
TryingOn.exe
```

恢复完成后，再双击运行：

```text
TryingOn.exe
```

## 文件说明

- `时装ing-win-x64/`：完整 Windows x64 桌面版程序文件夹。
- `时装ing-win-x64/restore-exe.bat`：下载 ZIP 后先双击它，恢复启动程序。
- `时装ing-win-x64/exe-parts/`：启动程序分包，避免 GitHub 普通仓库 100MB 单文件限制。
- `时装ing-win-x64/TryingOn.exe`：恢复后出现的启动程序。
