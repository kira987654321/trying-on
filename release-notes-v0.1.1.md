# Trying On 0.1.1｜时装ing出图工作台优化

这是 `Trying On / 时装ing` 的 v0.1.1 更新。本次发布不覆盖 v0.1.0，重点是把出图流程整理成更清晰的三栏工作台，并补充真实生成样张。

## 下载方式

普通用户请下载本 Release 里的：

`trying-on-0.1.1-win-x64.zip`

不要下载仓库首页绿色 `Code -> Download ZIP`，那个是源码包，不是可直接运行的软件。

解压后进入 `时装ing-win-x64`，双击 `时装ing.exe` 即可打开。请保留整个解压后的文件夹，不要只拷贝单独 exe。

## 本次更新重点

- 新版三栏出图工作台：左侧上传素材，中间选择模板组，右侧确认生成状态。
- 上传区改为产品摄影台风格，主商品图展示更清楚，适合上传衣服大图。
- 右侧补充素材卡支持 `Logo/标识`、`辅料细节`、`工艺实拍`。
- 界面只保留清晰的出图工作台体验，顶部入口固定为 `生成` 和 `历史结果`。
- 模板组固定为两个方向：`模板组一 / 商务冰感质感`、`模板组二 / 浅色弹力质感`。
- 点击模板组后立即展示当前组 5 张参考预览。
- 点击模板预览可以打开大图画廊查看细节。
- 历史结果页可查看本机生成记录和生成进度。
- 真实测试中，一组 5 张可以按用户配置自动调用 API 生成。

## API 配置说明

本版本暂不固定服务商 URL。用户仍然在软件 `设置` 中自行填写：

- `Base URL`
- `API Key`
- 模型名

API Key 不会上传到 GitHub，也不会写入公开仓库。配置保存在用户本机。
